\doxysubsection{Detailed Description}
A class for managing video objects, extending the \doxylink{class_multimedia_object}{Multimedia\+Object} class. 

This class represents a video, including its duration along with the basic multimedia attributes inherited from \doxylink{class_multimedia_object}{Multimedia\+Object}.
