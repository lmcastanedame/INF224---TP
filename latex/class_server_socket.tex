\doxysubsection{Detailed Description}
TCP/\+IP IPv4 server socket. Waits for requests to come in over the network. TCP/\+IP sockets do not preserve record boundaries but \doxylink{class_socket_buffer}{Socket\+Buffer} solves this problem.
