\doxysubsection{Detailed Description}
A class that extends \doxylink{class_video}{Video} to manage films, including chapter information. 

This class provides functionalities to handle film-\/specific attributes such as chapters, in addition to inheriting common video attributes from the \doxylink{class_video}{Video} class.
